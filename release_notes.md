## 软件功能

**采购助手** 是一款专为食品行业包装采购设计的桌面管理工具，基于 Python + CustomTkinter 构建，莫兰迪暖色调 UI，支持 Win10/11。

### 核心模块

| 模块 | 功能 |
|------|------|
| 数据看板 | 月度采购金额趋势图、待催款统计、卡片式概览 |
| 采购下单 | 物料采购下单，自动匹配供应商和报价 |
| 物料查询 | 按物料编号/名称模糊搜索，支持 CSV 导入导出 |
| 催款管理 | 采购款项到期提醒与催款记录追踪 |
| 垫付管理 | 个人垫付采购款记录与统计 |
| 差旅报销 | 差旅费用录入、报销状态跟踪 |
| 备忘录 | 日常待办事项与工作笔记 |
| 三方比价 | 多供应商价格对比，生成比价表 |
| 报价单 | 产品报价单生成与供应商管理，支持 Excel 导出 |
| 合同管理 | 采购合同解析（.docx），自动提取关键字段 |
| 产品 BOM | 产品物料清单（BOM）查询与管理 |
| 包装明细 | 产品包装规格管理与报价模板导出 |
| 系统设置 | 主题切换（亮色/暗色）、数据路径、自动更新检查 |

### 技术特点

- 本地 SQLite 数据库，数据安全可控
- 自动更新检查（GitHub Releases）
- 系统托盘常驻，最小化不退出
- 支持导入导出 Excel/CSV 模板
