# 采购助手 使用说明

> 版本 V2.2.5 | eastseo

---

## 一、界面概览

- **折叠按钮**：标题栏最左侧，点击可隐藏/展开左侧导航栏
- **窗口控制**：标题栏右侧 `─` 最小化 / `□` 最大化 / `✕` 关闭
- **窗口缩放**：鼠标移至窗口边缘，光标变为箭头后拖拽即可调节大小
- **窗口拖动**：标题栏空白区域按住拖拽

---

## 二、功能模块

### 📊 看板总览
首页仪表盘，展示核心 KPI 指标卡和最新动态流。可按类型筛选动态，点击刷新按钮更新数据。

### 📦 下单管理
管理包材采购订单。支持**新增**、**上传**合同文件（自动解析）、**导出** Excel、**归档**记录。

### 📋 产品报价单
生成产品报价单。操作流程：**添加**产品 → 设置**需方**信息 → **+供方**添加供应商 → **预览**报价单 → **导出** Excel。

### ⚖ 三方比价
比价记录管理。填写产品信息（品名/项目号/材质结构/规格尺寸）和三家供应商价格，确认后生成比价记录，支持**导出** Excel。

### 📝 合同管理
自动生成采购合同。操作流程：
1. 填写合同编号和下单物料
2. 选择签订日期
3. 点击 **+供方** 管理供应商信息
4. 点击 **需方** 配置甲方信息
5. 添加产品信息（支持多行）
6. 点击 **生成** 导出 Word 合同

### 🏭 厂家管理
供应商信息库。支持**新增**、**导出**、**导入**供应商数据。

### 📑 物料台账
物料采购记录查询。支持按年份、月份、供应商等条件筛选，**导出** Excel。

### 🔧 BOM管理
产品 BOM 结构查询。

### 💰 垫付与报销
被催款记录管理。支持**新增**催款记录、**导出**和**导入**数据。

### 💳 垫付管理
采购垫付记录。支持**新增**、**导出**、**导入**。

### ✈ 差旅管理
差旅报销记录。支持**新增**、**导出**、**导入**。

### 📌 待办备忘录
日常待办事项管理，支持新增、编辑、删除，自动记录创建时间。

## 三、通用操作

| 操作 | 说明 |
|------|------|
| 数据导出 | 各页面右上角 **导出** 按钮，导出为 Excel |
| 数据导入 | **导入** 按钮，支持 xlsx 格式批量导入 |
| 表格排序 | 点击表头可按该列排序 |
| 行内编辑 | 部分表格支持双击单元格直接编辑 |

## 四、系统设置

通过左侧导航 → 底部 ⚙ 图标进入：

- **外观设置**：切换明亮/暗色主题
- **数据管理**：数据库备份与恢复
- **启动设置**：开机自启、最小化到托盘
- **系统设置**：托盘图标、窗口位置记忆
- **数据备份**：手动备份/恢复 SQLite 数据库
- **软件介绍**：版本信息与更新检查

> 💡 关闭窗口时应用会**隐藏到系统托盘**而非退出，右键托盘图标可完全退出。

---

## 五、数据存储

所有数据存储在本地 SQLite 数据库：

```
C:\Users\<用户名>\采购管理系统数据\procurement.db
```

建议定期在**系统设置 → 数据备份**中备份数据库。
