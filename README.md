## 核心模块详解

### 📈 采购看板

- **KPI 核心指标卡片**：一行四列圆角卡片，展示本月采购金额、订单数量、应付金额、待处理事项
- **最新活动动态**：实时推送最近的操作记录（下单、催款、垫付、报销、备忘录）
- **数据来源**：自动聚合 packaging_orders / collection_reminders / purchase / travel / memos 五个数据表

### 📦 物料下单

- 新增、编辑、删除包装物料采购订单
- 支持上传采购合同（.docx），自动解析合同中的物料信息
- CSV 批量导入物料数据
- 订单状态追踪（待审核 / 已确认 / 已完成）
- **V2.0 新增**：操作列删除/归档功能修复（col_idx 索引修正）

### 📑 报价管理

- 输入产品包装信息（品名、项目号、材质、规格、数量），自动生成报价单
- 内置 Excel 报价模板，支持一键导出
- 报价历史记录管理

### 🤝 三方比价

- 记录三家供应商报价信息（单价、交期、付款方式等）
- **阶梯拆分**：支持按数量阶梯拆分多行记录，产品信息（品名/项目号/材质/规格）自动合并
- 自动生成比价 Excel 表格，文件名格式：`品名_项目号_供应商名_YYYY-MM-DD_比价表.xlsx`
- 最终做货供应商注入至表格正确位置（G12），与签字栏平行对齐
- 导出前自动清除比价表模板外的多余 sheet
- 导出时自动处理文件占用（PermissionError 捕获 + 友好提示）

### 📄 合同生成

- 基于 docx 模板自动生成采购合同
- 自动填充甲方信息（从 party_a.json 读取）
- 支持自定义模板和签字区域

### 🏭 厂家管理

- 供应商信息 CRUD 管理
- 维护供应商联系人、电话、地址等档案信息
- 支持导入/导出供应商数据

### 🔎 物料查询

- 多条件筛选查询（品名、项目号、规格、材质等）
- CSV 模板导入和 Excel 导出
- 支持数据批量操作

### 📋 成品 BOM

- 产品物料清单（Bill of Materials）管理
- Excel 导入导出，支持批量编辑
- BOM 结构与关系维护

### 💰 应付管理

- 催款记录管理（供应商、金额、到期日、状态）
- 款项回收进度跟踪
- 支持按状态筛选（待催收 / 已催收 / 已到账）

### 💳 垫付管理

- 采购垫付款项记录（金额、事由、日期）
- 关联报销流程进行闭环管理
- 垫付状态追踪（待报销 / 已报销）

### 🧾 差旅报销

- 差旅费用明细录入（交通、住宿、餐饮等）
- 汇总统计与多维度筛选
- 支持按日期范围、类别筛选

### 📝 待办事项

- 工作备忘录增删改查
- 分类管理与关键字搜索
- 一目了然的待办列表

### ⚙️ 系统设置

- **外观设置**：亮色/暗色模式切换、莫兰迪暖色调配色、主题色彩自定义
- **数据管理**：自定义数据存放目录，支持数据库迁移
- **启动设置**：开机自启（Windows 注册表）、系统托盘常驻
- **软件介绍**：版本信息、功能模块概览、技术栈说明
- **版本更新**：启动时自动检测 GitHub Releases，有新版弹窗提醒

## 技术架构

| 层级 | 技术选型 |
|------|----------|
| 语言 | Python 3.12 |
| GUI 框架 | CustomTkinter 5.2 |
| 数据库 | SQLite（本地文件存储，零配置） |
| 图像处理 | Pillow（PIL） |
| 文档处理 | python-docx（合同）、openpyxl / lxml（Excel、比价表） |
| 打包分发 | PyInstaller 6.20（单文件 EXE，深度精简 39MB） |
| 版本管理 | Git + GitHub Releases |
| 系统托盘 | pystray |
| 自动更新 | GitHub API（后台异步检查） |

### 打包优化（深度精简）

V2.0 采用 PyInstaller excludes 策略，主动排除项目中未使用的 AI/ML 等大体积依赖包（torch、scipy、transformers、pandas 等 29 个包），将 EXE 从原始 298MB 缩减至 **39MB**（缩减 82.6%），大幅提升下载和启动速度。

## 快速开始

### 运行环境

- Windows 10 / 11（64 位）
- 无需安装 Python 环境，下载 EXE 即可运行

### 直接运行（推荐）

1. 前往 [GitHub Releases](https://github.com/eastseao/procurement-system/releases) 下载 `采购助手V2.0.2.exe`
2. 双击运行，数据自动存储在 `%USERPROFILE%\采购管理系统数据\`

### 从源码运行

```bash
# 克隆仓库
git clone https://github.com/eastseao/procurement-system.git
cd procurement-system

# 安装依赖
pip install customtkinter Pillow pystray python-docx openpyxl lxml

# 运行
python main.py
```

### 打包 EXE

```bash
# 使用 Python 3.12
pip install pyinstaller
pyinstaller --clean 采购助手V2.0.2.spec
# 输出: dist/采购助手V2.0.2.exe (~39MB)
```

## 设计特色

- **莫兰迪暖色调**：主色 `#D4917A`（陶土色），视觉柔和温润，适合长时间使用
- **可折叠侧边栏**：采购/数据/财务/工具四组导航，支持分组折叠和整个侧边栏收起
- **亮色/暗色双主题**：通过设置页面一键切换，重启生效
- **系统托盘常驻**：关闭窗口最小化至托盘，不占用任务栏空间
- **开机自启**：Windows 注册表方式，无需额外配置
- **单实例保护**：Windows Mutex 互斥锁，防止重复启动
- **异常日志**：运行错误自动记录到 error.log，便于排查
