## 系统概述

### 系统简介

采购管理系统是一款面向企业采购部门的桌面应用程序，基于 Python + CustomTkinter + SQLite 技术栈开发。系统采用莫兰迪暖色调 UI 设计，支持 Windows 10/11 原生适配，提供从物料管理、供应商管理到采购垫付、差旅报销的全流程数字化管理。

### 主要功能

• 物料管理：物料信息查询、新增、编辑、导入导出

• 物料下单：创建包装物料订单，支持 Excel 导入

• 报价单管理：产品报价单创建、编辑、导出 Excel

• 供应商管理：供应商信息维护、资质管理

• 催款记录：付款催收记录管理

• 采购垫付：采购垫付款项记录

• 差旅报销：差旅费用报销管理

• 备忘录：工作备忘记录

### 技术栈

| 技术层 | 技术选型 |
|--------|-----------|
| 前端框架 | CustomTkinter (CTk) |
| 后端语言 | Python 3.12+ |
| 数据库 | SQLite 3 |
| 打包工具 | PyInstaller |
| UI 配色 | 莫兰迪暖色调 |

## 打包说明

### 环境要求

• Python 3.12+

• CustomTkinter 5.2+

• PyInstaller 6.0+

• Pillow (PIL) 10.0+

### 打包步骤

1. **安装依赖**：

   ```
   pip install customtkinter Pillow openpyxl pyinstaller
   ```

2. **执行打包**：

   ```
   pyinstaller "采购助手V1.3.spec"
   ```

3. **打包完成后**，exe 文件位于 `dist/采购助手.exe`

### 注意事项

• spec 文件中的 `name='采购助手'`，不显示版本号

• `console=False`，不显示控制台窗口

• `icon='assets/同仁堂企业LOGO2.ico'`，使用同仁堂企业图标

• `datas` 中包含所有资源文件（assets/ 目录下的所有文件）

## 使用说明

### 首次使用

1. 双击"采购助手.exe"启动程序

2. 程序会自动在用户目录下创建"采购管理系统数据"文件夹

3. 所有数据文件（procurement.db、settings.txt、error.log）都存放在这个文件夹中

### 数据管理

1. 打开设置页面，进入"数据管理"分类

2. 点击"浏览..."按钮，选择新的数据存放位置

3. 点击"保存设置"，程序会自动迁移已有数据库文件到新位置

### 导航栏使用

• 点击导航栏顶部的 Logo，可以跳转到堂训页面

• 点击导航栏中的按钮，可以切换到对应的功能页面

• 点击底部的折叠按钮，可以展开/收起导航栏

## 附录：API 说明

### Database 类

Database 类封装了所有数据库操作，支持自定义数据目录。

#### 初始化

```python
db = Database(data_dir=None)
```

• `data_dir`：可选参数，指定数据库文件存放目录

• 如果不指定，默认使用 `~/采购管理系统数据/`

#### 主要方法

• `fetch_all(sql, params=())`：执行查询，返回所有结果

• `fetch_one(sql, params=())`：执行查询，返回第一条结果

• `execute(sql, params=())`：执行 INSERT/UPDATE/DELETE

• `executemany(sql, seq_of_params)`：批量执行

• `create_tables()`：创建所有数据表

### 设置管理

设置通过 `load_settings()` 和 `save_settings(settings)` 函数管理。

#### 设置项

• `theme`：主题（light/dark）

• `data_dir`：数据存放位置

• `startup_sections`：启动时需要打开的页面

• `window_geometry`：窗口大小和位置
